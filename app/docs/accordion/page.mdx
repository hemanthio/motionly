## Component API

### FAQAccordion

| Prop      | Type          | Default              | Description                                  |
| :-------- | :------------ | :------------------- | :------------------------------------------- |
| faqItems  | Array FAQItem | Predefined FAQ items | Array of FAQ items with question and answer. |
| className | string        | ''                   | Optional CSS class for additional styling.   |

### FAQ Item Interface

| Property | Type   | Description                   |
| :------- | :----- | :---------------------------- |
| question | string | The question text to display. |
| answer   | string | The answer text to display.   |
