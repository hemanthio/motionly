## Component API

### FAQAccordion

| Prop      | Type           | Default              | Description                                  |
| :-------- | :------------- | :------------------- | :------------------------------------------- |
| faqItems  | Array FAQItem | Predefined FAQ items | Array of FAQ items with question and answer. |
| className | string         | ''                   | Optional CSS class for additional styling.   |

### FAQ Item Interface

| Property | Type   | Description                   |
| :------- | :----- | :---------------------------- |
| question | string | The question text to display. |
| answer   | string | The answer text to display.   |

## Customization

The accordion component can be easily customized by:

1. **Modifying FAQ Items**: Update the `faqItems` array with your own questions and answers
2. **Styling**: Customize colors, spacing, and animations through Tailwind classes
3. **Animation**: Adjust the fade-in animation duration and easing in the CSS
4. **Icons**: Replace ChevronUp/ChevronDown with your preferred icons

## Features

- ✅ Smooth expand/collapse animations
- ✅ Hover effects on accordion items
- ✅ Customizable FAQ content
